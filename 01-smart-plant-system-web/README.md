# Smart Plant System Dashboard

Ini adalah repo untuk Web dashboard IoT proyek Smart Plant System

## Arsitektur

Website ini adalah dashboard IoT untuk proyek Smart Plant System. Website ini akan memuat monitoring terkait suhu, kelembaban tanah, dan lain-lain. Selain itu, lewat website kita bisa melakukan logika kontrol atau otomasi untuk menyiram tanaman.
