# Catatan Firebase

Firebase adalah platform pengembangan aplikasi mobile dan web yang menyediakan berbagai layanan, termasuk database, autentikasi, hosting, dan banyak lagi.

Firebase memiliki dua jenis database:
1. Realtime Database: Database NoSQL yang menyimpan data dalam format JSON dan menyinkronkan data secara real-time ke semua klien yang terhubung.
2. Firestore: Database NoSQL yang menyimpan data dalam format dokumen dan koleksi, serta menyediakan fitur query yang lebih canggih.

Dalam proyek ini, saya menggunakan firebase sebagau database untuk menyimpan data sensor dan kontrol perangkat. Kemudian, saya membuat website untuk menampilkan data sensor dan kontrol perangkat yang mengambil data dari firebase dan mengirim data ke firebase.
